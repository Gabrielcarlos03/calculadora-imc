# 🧮 Calculadora de IMC

Calculadora de Índice de Massa Corporal desenvolvida com HTML, CSS e JavaScript puro. Projeto estruturado com organização profissional de arquivos e código separado por responsabilidade.

---

## 📋 Sobre o Projeto

Este foi o primeiro projeto em que apliquei JavaScript com manipulação de DOM de forma prática — transformando uma lógica matemática em uma interface funcional e interativa para o usuário.

O foco foi construir algo simples, mas com estrutura de código organizada e design próprio.

---

## ⚙️ Funcionalidades

- Cálculo do IMC a partir do peso e altura informados
- Classificação do resultado (abaixo do peso, normal, sobrepeso, obesidade)
- Interface limpa e responsiva
- Feedback visual do resultado para o usuário
